Detailed Description

Interface class for the mult kernel.

mult multiplies two input arrays together, element-wise. For more details, look at the kernel's documentation.

Note
The mult kernel is available in kernels/math_kernels.cl.
The class creates its own buffers. If you would like to provide your own buffers, call get to get references to the placeholders within the class and assign them to your buffers. You will have to do this strictly before the call to init. You can also call get (after the call to init) to get a reference to a buffer within the class and assign it to another kernel class instance further down in your task pipeline.

The following input/output OpenCL memory objects are created by a Mult instance:

Name Type Placement I/O Use Properties Size
H_IN_A Buffer Host I Staging CL_MEM_READ_WRITE \(3*width*height*sizeof\ (cl\_float)\)
H_IN_B Buffer Host I Staging CL_MEM_READ_WRITE \(3*width*height*sizeof\ (cl\_float)\)
H_OUT Buffer Host O Staging CL_MEM_READ_WRITE \(3*width*height*sizeof\ (cl\_float)\)
D_IN_A Buffer Device I Processing CL_MEM_READ_ONLY \(3*width*height*sizeof\ (cl\_float)\)
D_IN_B Buffer Device I Processing CL_MEM_READ_ONLY \(3*width*height*sizeof\ (cl\_float)\)
D_OUT Buffer Device O Processing CL_MEM_WRITE_ONLY \(3*width*height*sizeof\ (cl\_float)\)
